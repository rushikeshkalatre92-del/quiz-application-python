# Quiz Application in Python

## Project Overview
This is a simple Quiz Application developed using Python. The application asks multiple-choice questions and calculates the final score based on user answers.

## Features
- Multiple-choice quiz questions
- User input handling
- Score calculation
- Final result display

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code
2. Run the Python file
3. Answer the quiz questions
4. View the final score

## Project Outcome
This project helped in understanding Python basics, loops, conditions, and user input handling.
